## porcupine-angular

This is the Angular workspace for building the `porcupine-angular` package.

## Install

Run `yarn` to install the Angular workspace dependencies:

```console
yarn
```

## Build

To build the project:

```console
yarn build
```

## Lint and Format

Format code with Prettier:

```console
yarn format
```

Lint code with ESLint:

```console
yarn lint
```
